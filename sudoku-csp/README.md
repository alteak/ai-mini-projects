# Sudoku Solver using Constraint Satisfaction Problems

This mini-project applies Constraint Satisfaction Problem (CSP) concepts to solve Sudoku.

## Description

The Sudoku puzzle is modeled as a CSP, where each cell is a variable, the possible digits are the domain, and the Sudoku rules are the constraints.

## Concepts Covered

- Variables
- Domains
- Constraints
- Backtracking search
- Constraint propagation

## What I Learned

- How to model a problem using CSP principles
- How constraints reduce the search space
- How backtracking can be used to find valid solutions

## Technologies Used

- Java
- Constraint Satisfaction Problems
- Backtracking
