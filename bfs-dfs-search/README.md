# BFS and DFS Search

This mini-project explores uninformed search algorithms, mainly Breadth-First Search (BFS) and Depth-First Search (DFS).

## Description

The goal of this project was to understand how basic search algorithms explore a state space and find possible solutions to a problem.

## Algorithms Covered

- Breadth-First Search (BFS)
- Depth-First Search (DFS)

## What I Learned

- How to represent a problem as states and transitions
- How BFS explores nodes level by level
- How DFS explores paths deeply before backtracking
- Differences between completeness, memory usage, and search strategy

## Technologies Used

- Java
- Algorithm design
- Problem solving
