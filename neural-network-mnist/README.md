# Neural Network for Handwritten Digit Classification

This mini-project introduces Artificial Neural Networks for classification tasks.

## Description

The project focuses on using a neural network to classify handwritten digits, based on concepts from supervised learning and deep learning.

## Concepts Covered

- Supervised Learning
- Artificial Neural Networks
- Training and testing
- Classification
- Backpropagation
- Model accuracy

## What I Learned

- How neural networks learn from data
- How classification models are trained and evaluated
- How backpropagation helps update model weights

## Technologies Used

- Python
- Machine Learning
- Neural Networks
