# Genetic Algorithm for the Traveling Salesman Problem

This mini-project applies Genetic Algorithms to an optimization problem, such as the Traveling Salesman Problem.

## Description

The project explores how evolutionary computation can be used to search for good solutions in complex optimization problems.

## Concepts Covered

- Population
- Fitness function
- Selection
- Crossover
- Mutation
- Generations

## What I Learned

- How Genetic Algorithms imitate natural evolution
- How candidate solutions can be improved over multiple generations
- How optimization problems can be solved using metaheuristic approaches

## Technologies Used

- Java
- Evolutionary Computing
- Optimization
