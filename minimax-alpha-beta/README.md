# Minimax and Alpha-Beta Pruning

This mini-project explores adversarial search algorithms used in game-playing AI.

## Description

The goal of this project was to understand how an AI agent can choose optimal moves in a competitive environment.

## Concepts Covered

- Game trees
- Minimax algorithm
- Alpha-Beta Pruning
- Evaluation functions
- Adversarial decision-making

## What I Learned

- How Minimax evaluates possible future moves
- How Alpha-Beta Pruning improves efficiency
- How AI strategies can be applied in two-player games

## Technologies Used

- Python
- Artificial Intelligence
- Game Theory
